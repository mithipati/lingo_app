# Lingo App

### About

Application for sharing lingo in various groups. Add your own words, make your own memes, and follow groups in a fun and creative atmosphere.

### Technologies Used

* Backend: Ruby on Rails
* Frontend: JavaScript, Twitter Bootstrap
* APIs: Meme Generator
